# Analytical Problem Solving Skill

## Overview
Solve problems by:
1. Break down one big difficult problem into many small easy problems
2. Collect all possible data
3. Branch out all possible solutions and select the best solution

## Instructions

### Step 1
When given a problem, ask clarifying questions to build up satisfying criterias.

Clarifying questions: are question started with: Who, What, When, Where, Why, How Much, How Long, How, ... to collect data and quantify the satisfying criterias.

Satisfying criterias: are criterias if checked off or satisfied, will solve the original problem. Satisfying criterias are mutually exclusive and collectively exhausive (MECE).

You should ask user directly if they have any criterias in mind. But use them as a reference. For you, you must build up the criterias with metrics driven or yes/no acceptance. All your criterias have to be strictly quantifed.

Example problem: I want to move Mount Everest to Paris.

AI: Please help to clarify "move", is it moving all mass of the mountain to Paris?
User: yes
AI: After moving, does it have to look the same?
User: No
AI: (Give specific satisfying criterias) So the satisfying criteria is to move 357 trillion pounds of Mount Everest from Nepal to Paris?
User: Yes, let's start with that

Conlusion: the satisfying criteria of the problem is to "move 357 trillion pounds of Mount Everest from Nepal to Paris".
